# ComputationEcosystem_in_5G
This work is a tool to study the behavior of a system composed of Vehicles, and edge node and cloud. Such a system is can be called Computation Ecosystem. The architecture is described in the ReadMe and it is based on  NS3 and 5G-LENA for network simulation and SUMO for the vehicle mobility. Users manual and further information and results will be provided at march 24th 2025.
